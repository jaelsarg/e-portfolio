Magic the Gathering

I developed an individual web application that recreates the flow of opening Magic: The Gathering booster packs and building a deck from the generated cards. The project was built with HTML, CSS, and JavaScript, without using frameworks or external libraries, and focused on dynamic content rendering, form validation, filtering, and interactive state management.

The application includes a subscription form with validation, a booster-opening experience where cards are revealed in the correct rarity order, a deck-building interface with searchable and color-based card filters, automatic placement of cards into mana-value zones, a statistics page, and a final confirmation screen. The project also required the use of pre-defined data and configuration files, strict code structure, and disciplined Git workflow.
