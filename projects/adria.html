Adria

I worked on a group project set in a fictional future where Earth has been rebuilt after a global disaster. The goal was to design a startup and develop a software product that improves life in Adria, the first colony on Earth, while taking into account strict world-building rules, sustainability constraints, and different settlement contexts.

The project combined business, UX, analysis, and software development. We had to produce a business case, a marketing website, a full analysis document, a client/server proof of concept, and a final investor pitch, while also planning the work in GitLab and documenting daily standups.

For the concept itself, the system had to be innovative, collect and store user data, include a visualisation component, and give people a reason to keep using it. The assignment also required personas, user stories, flow charts, C4 diagrams, ERDs, interactive Figma wireframes, user testing, ethical analysis, and security risk assessment.

Technically, the proof of concept had to use a Vue.js client with Vuetify, a Java server built with Vert.x, OpenAPI documentation, and a SQLite database. The project also encouraged extra features such as push notifications, graphs, maps, and other advanced interactions depending on the chosen concept
