Labyrinth

I worked on a team project that involved building a digital version of the board game Labyrinth using a web client and a Java server. The project was split into two phases: first, developing the HTML, CSS, and JavaScript client against an existing API, and then replacing that backend with our own server built in Java with Vert.x.

The game had to follow the official Labyrinth rules while also supporting digital-specific features such as bearer authentication, secure token handling, JavaScript polling, and strict client-server separation. In addition to the implementation, the project required wireframes, documentation of the authentication and authorization scheme, retrospectives, peer assessments, and a screencast demonstrating the final functionality.
